### UdaciFitness

UdaciFitness is an app for tracking triathlon (running, swimming, cycling).lismo). 

### Instalation

```
clone git https://github.com/gilmarvoge/UdaciFitness.git

#run
-cd UdaciFitness
-yarn install
-yarn start

-use app Expo to emulate android/iOS
```
